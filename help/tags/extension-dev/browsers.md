# サポートされているブラウザー

Adobe Experience Platform のタグのランタイム（クライアントの Web サイトに配信）は、複数のブラウザーと互換性があります。

- Chrome（最新）
-  Safari（最新）
- Firefox（最新）
- Edge（最新）
- iOS Safari（最新）
- Android Chrome（最新）

ランタイムに含まれる拡張機能コードも、これらのブラウザーのサポートに努める必要があります。

データ収集 UI は、次のブラウザーをサポートするように設計されています。

- Chrome（最新）
-  Safari（最新）
- Firefox（最新）
- Edge（最新）

データ収集 UI 内に表示される拡張機能コードも、これらのブラウザーのサポートに努める必要があります。
