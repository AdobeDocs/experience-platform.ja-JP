# 変数を更新

**[!UICONTROL Update variable]** アクションを使用すると、[ 変数データ要素 ](../data-element-types.md#variable) に対して部分的または増分的な変更を加えることができます。 このアクションを使用してオブジェクトを作成し、後で [[!UICONTROL Send event]](send-event.md) アクションで参照できるようにすることができます。 データ要素を入力し、XDM オブジェクト内のプロパティに割り当てることはほとんどのユースケースに適しています。このアクションでは、ルール条件に基づいて、様々なデータ要素に条件付きでプロパティを設定できます。

このアクションを使用する前に、変数データ要素を作成済みにする必要があります。 変更する変数データ要素を選択すると、エディターが表示され、このアクションに必要なフィールドを設定できます。

![ アクション設定インターフェイス内の変数を更新アクションのスクリーンショット ](../assets/update-variable.png)

エディターで使用される XDM スキーマは、変数データ要素内で選択されたスキーマと一致します。 オブジェクトを展開して目的のプロパティを選択することで、オブジェクトの 1 つまたは複数のプロパティを設定できます。 例えば、以下のスクリーンショットでは、`producedBy` プロパティは、データ要素 `%Produced by data element%` に設定されています。

![ 更新されたプロパティを示すアクション設定インターフェイスのスクリーンショット ](../assets/update-variable-set-property.png)

XDM オブジェクトではなくデータオブジェクトを使用する変数データ要素を選択した場合、使用できるフィールドは、データ要素の設定時に選択した製品によって異なります。 例えば、Adobe Analyticsとフィールドを含むデータオブジェクトを作成した場合、この UI で変数データ要素を選択すると、Adobe Analytics固有の入力が可能なフィールドが提供されます。

![ データオブジェクトに基づく変数データ要素を示すアクション設定インターフェイスのスクリーンショット ](../assets/variable-data-element-data.png)
