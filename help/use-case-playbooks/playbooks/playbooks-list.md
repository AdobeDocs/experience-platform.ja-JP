# 使用可能なユースケースプレイブック

このページを読んで、リアルタイムの顧客データPlatformおよびAdobe Systemsジャーニーオプティマイザーで検出して使用できるすべてのプレイブックの概要を確認してください。

## Journey Optimizer のユースケースプレイブック {#optimizer-playbooks}

>[!CONTEXTUALHELP]
>id="platform_playbooks_list"
>title="使用可能なプレイブック"
>abstract="Real-time Customer Data Platform と Adobe Journey Optimizer のすべてのプレイブックのリスト。"

| プレイブック名 | 説明 |
| ------------- |  ----------- |
| ブラウジングの放棄：商品 | 商品をブラウジング後にウェブサイトやアプリを離れたファンに再度エンゲージメントしてもらうよう促します。 |
| 放棄されたブラウジング:製品 | 商品ブラウジング後にウェブサイトやアプリを離れた買い物客に再度エンゲージメントするよう促します。 |
| 放棄されたブラウジング:予約 | 潜在的な予約ブラウジングした後、ゲストがウェブサイトやアプリを離れた後に再度エンゲージするように促します。 |
| 放棄されたカート:商品 | 買い物かごにグッズを追加した後、ウェブサイトやアプリを離れたファンに再度エンゲージメントするよう促します。 |
| 放棄された買い物かご: 製品 | 買い物客が買い物かごに商品を追加した後、ウェブサイトやアプリを離れた後に再び反応するように促します。 |
| 放棄されたカート:予約 | ゲストがウェブサイトやアプリを離れた後、買い物かごに新しい予約を追加した後、再度エンゲージするように促します。 |
| アプリダウンロードCTAのチェックインリマインダー | 情報に簡単にアクセスできるように、チェックインしてアプリのダウンロードを促すようにゲストに促します。 |
| 宛先のプロモーション | 進行中の観光地プロモーションキャンペーンの間に新しい予約をするようにゲストを動かしてください。 |
| ゲームプロモーション | ファンに今後のゲームのチケットを購入するように促します。 |
| ファンのためのアプリ内ウェルカム | 新しいファンを歓迎します。 |
| ゲストのアプリ内ウェルカム | 新しいお客様をお迎えします。 |
| 買い物客向けのアプリ内ウェルカム | 新しい買い物客ようこそ。 |
| ゲーム中の商品購入を増やす | ファンがゲームサイトの近くにいる間に商品を購入するためにナッジします。 |
| 来店時の製品購入の増加 | 買い物客が店の近くにいる間に商品を購入するように微調整します。 |
| 滞在中の製品およびサービス購入の増加 | ゲストがプロパティにいる間に製品やサービスを購入するように促します。 |
| シングルゲームの購入を増やす | リピートシングルゲームの購入を増やすためにファンTarget。 |
| Postイベント調査とインセンティブプロモーション | インセンティブプロモーションを提供することで、投稿イベントサーベイに対応するようファンを誘導します。 |
| 旅行後の調査 | インセンティブプロモーションを提供することで、投稿旅行のサーベイに対応するようにゲストを誘導します。 |
| 製品プロモーション | 進行中の製品プロモーションキャンペーン中に買い物客に製品を購入するように促します。 |
| 購入確認:商品 | 新規商品購入ファン確認。 |
| 購入の確認: 製品 | 新製品購入買い物客確認。 |
| 購入確認：予約 | 新しい予約ゲストを確認します。 |
| ファンを再エンゲージ | コンバージョンを完了する前に放棄したファンをインテリジェントかつ責任を持って再エンゲージします。 インテリジェンスと共感を持って行動し、単なるリマインダーではなく、ファンのための体験を作成します。 |
| ゲストに再度働きかける | 予約を完了する前に、予約を放棄したゲストをインテリジェントかつ責任を持って再エンゲージします。 インテリジェンスと共感を持って行動し、単なるリマインダーではなく、ゲストのための体験を作成します。 |
| 買い物客に再度働きかける | 購入を完了する前に、購入を放棄した顧客をインテリジェントかつ責任を持って再エンゲージします。 知性と共感を持って行動し、単なるリマインダーではなく、顧客のエクスペリエンスを作成します。 |
| 送信 ファンへの誕生日メッセージ | 誕生日に特別なオファーでファンを迎えましょう。 |
| ゲストへの誕生日メッセージ送信 | 誕生日に特別なオファーでゲストを迎えましょう。 |
| 買い物客に誕生日メッセージ送信 | 彼らの誕生日に排他的なオファーで買い物客を迎える。 |
| 新しいファンをオンボーディングするためのメールを送信 | メールで新しく登録されたファンを歓迎します。 |
| オンボーディングの新しいゲストにメールを送信 | メールで新しく登録されたゲストを歓迎します。 |
| 新規買い物客のオンボーディングへの電子メール送信 | 新しく登録した買い物客に電子メールでようこそします。 |
| 旅行領収書 - チェックアウト | チェックアウト時に領収書をご提示ください。 |
| ミニパッケージ購入者へのシングルゲーム購入のアップセル | 単一のゲーム チケット購入者からゲーム チケット パッケージの購入者に変換するようにファンを誘導します。 |
| ようこそ新規チケットパッケージ購入者 | アプリダウンロードを促すようこそメールを送信することで、ファンエクスペリエンスを改善します。 |

## リアル時間顧客データPlatformの使用例プレイブック {#rtcdp-playbooks}

| プレイブック名 | 説明 |
| ------------- | ----------- |
| ブラウジングの中止：商品 – アクティベーション用 | 商品を閲覧した後に web サイトやアプリを離れた後にファンをリターゲットします。 |
| 閲覧を中止：製品 – アクティベーション用 | 商品ブラウジング後に Web サイトやアプリを離れた買い物客をリターゲティングします。 |
| 放棄されたブラウジング: 予約 - Activation向け | 潜在的な予約をブラウジングした後、ウェブサイトやアプリを離れたゲストを再ターゲットします。 |
| 放棄された買い物かご：商品 – アクティベーション用 | スポーツグッズを買い物かごに追加した後、web サイトやアプリを離れた後にファンをリターゲットします。 |
| 放棄された買い物かご：製品 – アクティベーション用 | 買い物かごに製品を追加した後、web サイトまたはアプリを離れた後に買い物客をリターゲットします。 |
| 放棄された買い物かご：予約 – 有効化のため | 買い物かごに新しい予約を追加した後、ウェブサイトやアプリを離れたゲストを再ターゲットします。 |
| 宛先の昇格 – アクティブ化の場合 | 進行中の旅行先プロモーション中に予約を行うようにゲストをターゲットにします。 |
| ゲームのプロモーション – アクティベーション用 | ファンをターゲットにして、今後のゲームのチケットを購入します。 |
| ゲーム中に商品の購入を増やす – アクティブ化 | ファンTarget、ゲームサイトの近くにいる間に商品を購入することができます。 |
| 店舗訪問中の製品購入を増やす（アクティベーション用） | 買い物客が店の近くにいる間に商品を購入するようにターゲットを設定します。 |
| 滞在中に製品やサービスの購入を増やしてアクティブ化 | 宿泊客が施設内で商品やサービスを購入することを目的とする。 |
| 製品プロモーション - Activation用 | 進行中の製品プロモーションキャンペーン中に、買い物客をターゲットに製品を購入させる。 |
| ファンに誕生日のメッセージを送信 – 有効化 | 誕生日にファンをターゲットに、「ハッピーバースデー」のメッセージと特別オファーを提供します。 |
| お客様にバースデーメッセージを送信して有効化 | 「誕生日おめでとうございます」というメッセージと特別オファーで、誕生日のお客様をターゲットに設定します。 |
| 送信 買い物客向けの誕生日メッセージ - Activation | 誕生日に買い物客をターゲットに、「Happy Birthday」メッセージと特別オファーを提供します。 |
