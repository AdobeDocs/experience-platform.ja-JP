# `personalization`

The `personalization` オブジェクトには、ユーザーに応じてユーザーエクスペリエンスを調整できるいくつかのプロパティが含まれています。 Adobe TargetまたはAdobe Journey Optimizerの実装では、表示されるコンテンツをユーザーごとにカスタマイズできる原動力となるので、特に役に立ちます。

方法に関するドキュメントを参照してください。 [パーソナライズされたコンテンツをレンダリング](../../personalization/rendering-personalization-content.md) を参照してください。
