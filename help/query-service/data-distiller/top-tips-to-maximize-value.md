# Adobe Experience Platform Data Distillerの価値を最大化するためのヒント

このページには、「OS656 - Adobe Experience Platform Data Distillerで価値を最大化するためのトップヒント」セッションで学んだことを適用できるサンプルデータセットが含まれています。 リアルタイム顧客プロファイルデータを強化して、Adobe Real-time Customer Data PlatformとJourney Optimizerの実装を高速化する方法を説明します。 このエンリッチメントでは、顧客の行動パターンに関する深いインサイトを活用して、エクスペリエンスの配信および最適化のためのオーディエンスを構築します。

ケーススタディを通じて、ユーザーの行動データを分析し、リーセンシー、フリークエンシー、通貨（RFM）モデルを作成します。これは、購入パターンに基づいて顧客をセグメント化するためのマーケティング分析手法です。
