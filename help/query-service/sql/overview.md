# SQLリファレンス

クエリサービスは、標準のANSI SQLを文やその他の制限付きコマンドに `SELECT` 使用する機能を提供します。 クエリサービスには、データ定義言語またはデータ操作言語機能は含まれていません。 サービスを介してクエリーされるデータは、主に不変のデータセットを含むExperience Data Lakeから取得されます。
