# SQLリファレンス

[!DNL Query Service] は、標準のANSI SQLを文や他の制限付きコマンドに使用する機能を提供し `SELECT` ます。 [!DNL Query Service] には、データ定義言語またはデータ操作言語機能は含まれません。 サービスを介してクエリーされるデータは、主に不変データセットを含む元 [!DNL Experience Data Lake]のものです。
