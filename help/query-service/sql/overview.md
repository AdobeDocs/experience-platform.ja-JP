# クエリサービス SQL リファレンス

[!DNL Query Service] には、標準の ANSI SQL を `SELECT` ステートメントおよびその他の制限付きコマンド [!DNL Query Service] には、データ定義言語やデータ操作言語の機能は含まれていません。 サービスを通じてクエリされるデータは、 [!DNL Experience Data Lake]：主に不変データセットを含みます。
