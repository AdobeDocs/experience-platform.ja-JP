# SQLリファレンス

クエリサービスは、標準ANSI SQLを文や他の制限付きコマンドに使用する機能を提供し `SELECT` ます。 クエリサービスには、データ定義言語またはデータ操作言語機能は含まれていません。 サービスを介してクエリーされるデータは、主に不変データセットを含むExperience Data Lakeから取得されます。
