# 同意分析とトラッキング

現在のマーケティング環境では、顧客の同意環境設定を理解し、尊重する必要があります。 Adobe Real-time Customer Data Platformは、マーケターが顧客の同意を分析して、信頼を構築し、プライバシー規制を遵守し、よりパーソナライズされたエクスペリエンスを提供する機能を提供します。

同意分析ガイドでは、Real-Time CDP データの様々なマーケティングユースケースに対する同意ダッシュボードの作成方法について詳しく説明します。 同意分析ダッシュボードを作成して、ユーザーの同意のトレンドを経時的に追跡する方法については、[&#x200B; 同意分析ダッシュボードガイド &#x200B;](../../dashboards/insights-use-cases/consent-analysis.md) を参照してください。
