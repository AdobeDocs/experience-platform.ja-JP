# API を使用したプロファイルのデータセットの設定

リアルタイム顧客プロファイルとAdobe Experience Platform ID サービスで使用するデータセットを有効にするチュートリアルでは、新しいデータセットを作成し、既存のデータセットを設定する手順を説明します。 このチュートリアルでは、データセットにデータを取り込み、プロファイルおよび ID サービスへのデータの取り込みが成功したことを確認する手順についても説明します。

このチュートリアルにアクセスするには、データセットに関するドキュメントの一部である [API を使用してプロファイルと ID のデータセットを有効にするチュートリアル ](../../catalog/datasets/enable-for-profile.md) を参照してください。
