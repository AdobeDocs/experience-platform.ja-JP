# リアルタイム顧客データプラットフォームのセグメントビルダー

[!DNL Real-time Customer Data Platform]はAdobe Experience Platform上に構築されており、[!DNL Experience Platform]の一部であるセグメントビルダーの全機能を利用できます。 セグメントビルダーのユーザーインターフェイス内で、新しいセグメントの作成、既存のセグメントへのアクセス、既存のセグメントの変更をおこなうことができます。

セグメントビルダーの詳細なガイドについては、『[セグメントビルダーユーザーガイド](../../segmentation/ui/segment-builder.md)』を参照してください。
