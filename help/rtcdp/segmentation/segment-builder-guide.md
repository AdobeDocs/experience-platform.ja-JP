# Real-time Customer Data Platformのセグメントビルダー

Adobe Experience Platform上に構築 [!DNL Adobe Real-Time Customer Data Platform] は、 [!DNL Experience Platform]. セグメントビルダーのユーザーインターフェイス内で、新しいオーディエンスの作成、既存のオーディエンスへのアクセス、既存のオーディエンスの変更をおこなうことができます。

セグメントビルダーの詳細なガイドについては、『[セグメントビルダーユーザーガイド](../../segmentation/ui/segment-builder.md)』を参照してください。
