# Platform Debugger のインストール

Adobe Experience Platform Debuggerは、Google Chrome ブラウザーにインストールできます。 以下のリンクに従って、ブラウザーに拡張機能をインストールします。

* [クロム](https://chrome.google.com/webstore/detail/adobe-experience-platform/bfnnokhpnncpkdmbokanobigaccjkpob)

Chrome 拡張機能をインストールすると、アイコン (![](images/start-icon.jpg)) が拡張機能バーに追加されます。 このアイコンを選択して、拡張機能を開きます。
