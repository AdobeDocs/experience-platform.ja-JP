# [!DNL Mailchimp] コネクタ

Adobe Experience Platform を使用すると、データを外部ソースから取得しながら、[!DNL Experience Platform] サービスを使用して、受信データの構造化、ラベル付け、拡張を行うことができます。アドビのアプリケーション、クラウドベースのストレージ、データベースなど、様々なソースからデータを取り込むことができます。

Experience Platform は、サードパーティのマーケティングオートメーションシステムからデータを取得する機能を備えています。マーケティングオートメーションプロバイダーのサポートには、[!DNL Mailchimp] が含まれます。

## IP アドレスの許可リスト

ソースをExperience Platformに接続する前に、地域固有の IP アドレスを許可リストに追加する必要があります。 詳しくは、[Experience Platformへの接続に対する IP アドレスの許可リストに加える](../../ip-address-allow-list.md) に関するガイドを参照してください。
