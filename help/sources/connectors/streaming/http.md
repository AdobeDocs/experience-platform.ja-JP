# HTTP API コネクタ

>[!IMPORTANT]
>
>HTTP API ソースのマッパー関数は、現在、API でのみサポートされています。ユーザーインターフェイスではサポートされていません。

Adobe Experience Platformを使用すると、データを外部ソースから取り込みながら、 [!DNL Platform] サービス。 アドビのアプリケーション、クラウドベースのストレージ、データベースなど、様々なソースからデータを取得することができます。
