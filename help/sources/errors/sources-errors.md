# ソースのエラーメッセージ

このドキュメントでは、ソースに関するエラーメッセージ、説明および推奨される解決策のカタログを示します。

## 一般的なエラー

| エラーコード | タイトル | 詳細なメッセージ |
| --- | --- | --- |
| `1000-400` | リクエストが正しくありません | リクエストが無効です。リクエストを確認し、再試行してください。 |
| `1001-401` | 許可されていません | ユーザーに権限がありません。管理者に連絡して、リソースへのアクセス権を取得してください。 |
| `1002-403` | 禁止されています | リクエストされた操作は禁止されています。リクエストされた操作を確認し、再試行してください。 |
| `1003-404` | リソースが見つかりません | リクエストされたリソースが見つかりません。提供されたリクエストを確認し、再試行してください。 |
| `1004-415` | サポートされていないメディアタイプ | 指定されたペイロード形式はサポートされていません。提供されたリクエストを確認し、再試行してください。 |
| `1005-500` | 内部エラー | 内部エラーが発生しました。再試行してください。問題が解決しない場合は、カスタマーサポートにお問い合わせください。 |
| `1006-408` | リクエストがタイムアウトしました | リクエストの処理中にエラーが発生し、リクエストがタイムアウトしました。再試行してください。問題が解決しない場合は、カスタマーサポートにお問い合わせください。 |
| `1007-400` | ヘッダーパラメーターが無効です | 無効なヘッダーパラメーター {headerName} を受け取りました。 ヘッダーパラメーターを確認し、もう一度試してください。 |
| `1008-401` | | 認証トークンが無効です | 認証トークンにこの組織へのアクセス権がないか、組織が存在しません。組織が存在することを確認するか、管理者に連絡してアクセス権を取得してください。 |
| `1009-403` | IMS 組織 ID が見つからないか空です | 組織 ID リクエストヘッダーがないか、空です。ヘッダー値を更新し、再試行してください。 |
| `1010-500` | 無効な詳細メッセージ | 詳細メッセージのパラメーターが正しく指定されていません。詳細メッセージのパラメーターを確認し、再試行してください。 |
| `1011-503` | サービスを利用できません | サービスは一時的に利用できません。再試行してください。問題が解決しない場合は、カスタマーサポートにお問い合わせください。 |
| `1012-504` | ゲートウェイがタイムアウトしました | ゲートウェイのタイムアウトが発生しました。再試行してください。問題が解決しない場合は、カスタマーサポートにお問い合わせください。 |
| `1013-412` | 事前条件が失敗しました | If-Unmodified-Since ヘッダーまたは If-None-Match ヘッダーで定義された条件が満たされません。確認してから再試行してください。 |
| `1014-400` | 無効なリクエストの不正な引数 | リクエストを処理できませんでした。{detailedMessage} |
