# 文字列関数

プロファイルクエリ言語(PQL)オファーは、文字列とのやり取りをより簡単にするための機能です。 その他のPQL関数の詳細については、「 [プロファイルクエリ言語の概要](./overview.md)」を参照。

## いいね！

この関 `like` 数は、文字列が指定したパターンと一致するかどうかを判断するために使用されます。

**形式**

```sql
{STRING_1} like {STRING_2}
```

| 引数 | 説明 |
| --------- | ----------- |
| `{STRING_1}` | チェックを実行する文字列です。 |
| `{STRING_2}` | 最初の式列と照合する文字列です。 式の作成に使用できる特殊文字は2つあります。 `%` と `_` <ul><li>`%` は、0文字以上の文字を表すために使用されます。</li><li>`_` は、1文字を表すために使用されます。</li></ul> |

**例**

次のPQLクエリは、「es」というパターンを含むすべての市区町村を取得します。

```sql
city like "%es%"
```

## 次で始まる

この関数 `startsWith` は、指定したサブ文字列を持つ文字列開始かどうかを判断するために使用されます。

**形式**

```sql
{STRING_1}.startsWith({STRING_2}, {BOOLEAN})
```

| 引数 | 説明 |
| --------- | ----------- |
| `{STRING_1}` | チェックを実行する文字列です。 |
| `{STRING_2}` | 最初の文字列内で検索する文字列です。 |
| `{BOOLEAN}` | チェックで大文字と小文字が区別されるかどうかを指定するオプションのパラメーターです。 デフォルトではtrueに設定されています。 |

**例**

次のPQLクエリは、名前の開始が「Joe」であるかどうかを大文字と小文字の区別で判断します。

```sql
person.name.startsWith("Joe")
```

## Contains

この関数 `contains` は、文字列に指定したサブ文字列が含まれているかどうかを判断するために使用されます。

**形式**

```sql
{STRING_1}.contains({STRING_2}, {BOOLEAN})
```

| 引数 | 説明 |
| --------- | ----------- |
| `{STRING_1}` | チェックを実行する文字列です。 |
| `{STRING_2}` | 最初の文字列内で検索する文字列です。 |
| `{BOOLEAN}` | チェックで大文字と小文字が区別されるかどうかを指定するオプションのパラメーターです。 デフォルトではtrueに設定されています。 |

**例**

次のPQLクエリでは、ユーザーの電子メールアドレスに文字列「2010@gm」が含まれているかどうかを、大文字と小文字の区別を考慮して判断します。

```sql

```

## 一致

この関 `matches` 数は、文字列が特定の正規式と一致するかどうかを判定します。 正規ドキュメントでのパタ [ーンの一致](https://docs.oracle.com/javase/8/docs/api/java/util/regex/Pattern.html) について詳しくは、この式を参照してください。

**形式**

```sql
{STRING_1}.matches(STRING_2})
```

**例**

次のPQLクエリは、名前の開始が「John」であるかどうかを、大文字と小文字を区別せずに判断します。

```sql
person.name.matches("(?i)^John")
```

## 正規式グループ

この関 `regexGroup` 数は、提供される正規の情報に基づいて特定の情報を抽出するために使用される式です。

**形式**

```sql
{STRING}.regexGroup({EXPRESSION})
```

**例**

次のPQLクエリを使用して、電子メールアドレスからドメイン名を抽出します。

```sql
emailAddress.regexGroup("@(\w+)", 1)
```
